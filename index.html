GIT: Global interaction tool Git is an source code management tool which is used for storing the code, important files, directories, images etc which uses distributed version control system mechanism which provides proper tracking and also benefit of storage cost optimization. ------------------------------------------------------------------- Why GIT ? ------------------------------------------------------------------- -------------------------------------------------- * What is version control system ? -------------------------------------------------- VCS - version control system is an mechanism which provides an unique identification for managing the data inside the files via commit_id which is an hexa decimal code in the form of 125/256 bytes using algorithm called "SHA key" secure hashing algorithm. eg: commit_id: xnshkjhg3bn5nn2njjdjdj0w2
